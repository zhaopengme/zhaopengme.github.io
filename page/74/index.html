烛光祈福

发表于 2008-05-20   |   分类于 闲言碎语   |  

今天和朋友一起组织了一场对四川灾区人民的烛光祈福活动，说实话，我还真没组织过什么大型活动，这次的活动我们也没有以任何组织或者单位举行，我们想的就是该给灾区人民做点事，下午在QQ群聊了几句，就一起商量做了起来。从下午3、4点说起到晚上9点组织开始，短短几个小时，我们考虑的想到不周全、不成熟。

请了学校的各个学生组织，可能都是组织惯了，我们说好各个组织9点钱都要来的，但是，可能是我没看见吧！当时在场的自发的学生都帮忙了，所以可能各个组织都被当作自发的学生一起了。

活动9：30开始了，大家点了蜡烛，默哀了三分钟，以后就是大家激情了，活动一次一次推向了爱国高潮。爱国高潮都是我们所期盼的，但是我们的爱国高潮对于我们的学校学生而言，真的有点怕了！我们该怎么收场啊！那时的气氛，大家都不愿意散的。

当时灯火通明，篮球杆上都爬满了人，公寓楼上的顶层都不知怎么的站出来了学生，好危险啊！最后有学校学生处的朋友来了，他们的速度也挺快的啊！

活动最后也是结束了，篮球场到处都是蜡烛，晚上都马上11点了，我们11点关楼门的，大家准备一起在第二天早起点清理，好好把操场清理了。

百度原来有这么多的新玩意-为盲人设计的搜索引擎

发表于 2008-05-01   |   分类于 软件工具   |  

今天点开百度产品首页，突然发现了一个“盲道”，盲道？新词吧！的确是个新词，点开看看，就是个搜索引擎，和首页一样，点开帮助看看介绍。



百度是这样说的。

盲道介绍

百是百度为了方便盲人方便的上网获取信息，经过长期大量的市场、技术调研而开发的。如果你有问题，请到盲道吧去跟更多人交流共享。

常见问题

1 百度盲道是什么？

百度盲道是百度提供的为方便视障人士上网阅读信息、参与社区互动、听歌、访问互联网上其他相关网站的特殊入口。



2 百度盲道有功能？

百度盲道集成百度产品线中的新闻、网页搜索、MP3、贴吧、知道、百科和网址导航等产品线。沿袭百度产品一贯的简单风格，同时在原来的各产品首页上去掉了很多会影响读屏软件内容，保持了最有效的访问内容的导航。



3 我在这里使用百度新闻跟www.baidu.com使用新闻有什么不同吗？

目前最大的不同在于，你通过这里访问到的新闻服务可以直接选择你需要的分类的进行浏览，也可以直接使用新闻订阅功能、个性化订阅等，而没有news.baidu.com进入以后的很多新闻直接浏览再按需要筛选的困扰。其他频道也是这样我们充分简化了登录这些产品的首页，使它主旨更明确更适合读屏软件的读取。



4 为什么才这点功能？

目前百度盲道只提供了盲人使用量较高的几个产品线的服务！如果您希望增加哪些服务也可以反馈给我们。



真郁闷，为盲人设计，感觉和我们正常人的是一样的啊！盲道的百度和我们正常用的百度，官方给的区别我还是没弄清楚！我在详细查查！



有个说法能直接给解释，百度的盲道优化了搜索界面，使读屏软件能够顺利地识别内容，进行语音提醒。 



我们平常人的电脑里没有装读屏软件，自然无法感觉 。



读屏软件，又是个新东东。原来区别就在这啊!郁闷吧!



一直以来在google和baidu之间我比较喜欢google，喜欢google的原因不用说，多了解一下google就知道了，从百度推出了百度hi以来，我又经常到百度去转转，发现百度的确有好多东西，虽然在某些方面比不上google，但毕竟是国内最大的搜索引擎，况且我还有点爱国的思想的。呵呵！



百度推出了盲道我就要说他了，推出了新产品也不到处炫耀炫耀，许多人都不知道的，而且他们的说明也不是很清楚，如果你不仔细研究一下盲道，那就一个和我们正常使用的百度一样的产品，会让人失望的。

用FinalData我们来恢复数据

发表于 2008-04-29   |   分类于 软件工具   |  




我们使用finalData来恢复数据，具体的步骤如下：

打开FinalData，选择“文件”-“打开”



出现如上的界面，我们选择要恢复的盘符，点“确定”。







此时会对你选择的盘进行文件的扫描。



Normal 0 7.8 磅 0 2 false false false MicrosoftInternetExplorer4 选择要搜索的簇范围，也就是从硬盘的某个地方到某个地方进行搜索，我们点“确定”就行。



根据范围的大小进行文件的搜索，我们等待时间的完成，范围大的话等待的时间会很长的。



扫描完了之后，就能出现我们需要恢复的文件了，我们在需要需要恢复的文件上点右键，选择“恢复”，选择需要的恢复的文件所保存的位置，点“保存”。



我们需要恢复的文件就被恢复了。呵呵！

恭喜你，你已经学会数据恢复了！哈哈
